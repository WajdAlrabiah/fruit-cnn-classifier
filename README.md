A Convolutional Neural Network (CNN) project that classifies images of fruits (mango and kiwi) using TensorFlow and Keras. This project demonstrates image preprocessing, model building, training, evaluation, and prediction with a simple CNN architecture.

---

## Features

- Load and preprocess fruit images from dataset.
- Resize images to 128x128 pixels and normalize input data.
- Split dataset into training and testing sets.
- Build a CNN model with multiple convolutional, batch normalization, pooling, and dropout layers.
- Train the model and visualize accuracy and loss.
- Evaluate model performance using accuracy, confusion matrix, and classification report.
- Predict class of new fruit images.
